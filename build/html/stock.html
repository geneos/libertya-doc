Módulo de Stock¶

Datos previos necesarios

  1. Almacenes y Ubicaciones
  2. Artículos
  3. Conjunto de Atributos (Opcional)
  4. Cargos

Proceso de gestión de stock

  1. Ingresos de Artículos al Stock
    1. Ingresos por Remito de Compras (Ver módulo de Compras).
    2. Ingresos por ajuste de Inventario Físico.
    3. Ingresos por Remito de Devolución de Clientes (Ver módulo de Ventas).
    4. Ingresos por Devolución de Materiales para Producción (Ver módulo de Producción).
    5. Cambio de Artículo.
  2. Movimientos de Stock entre Almacenes

    1. Movimiento de Inventario.

    2. Transferencia de Mercadería.
      1. Transferencia en dos Etapas.
      2. Movimiento en dos Etapas. Por ejemplo para envíos a garantía.

    3. Entradas y Salidas Simples. Por ejemplo para pérdidas.

  3. Bajas de Stock
    1. Baja por ajuste de Inventario Físico.
    2. Baja por Entrega de Materiales para Producción (Ver módulo de Producción).
    3. Baja lógica por Movimientos de Inventario a almacenes de descarte.
    4. Baja por Remito de Entrega a Clientes (Ver módulo de Compras).
    5. Baja por Remito de Devolución a Proveedores (Ver módulo de Ventas).
    6. Cambio de Artículo.

Configuración de cargos

Para algunas de las operaciones, se deben configurar elementos especiales, para detallar motivos y/o hacer afectación contable. Para esto se utilizan los Cargos.
