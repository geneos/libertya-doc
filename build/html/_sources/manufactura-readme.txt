**********************************
Manufactura - Notas de Instalación
**********************************

Antes de comenzar a utilizar el componente de Manufactura en Libertya, es importante tener en cuenta algunos aspectos que detallamos a continuación:

1. La descarga del componente está disponible en `Libertya MRP Geneos <http://libertya.geneos.com.ar/>`_.
2. Esta versión es BETA, lo que implica que puede contener bugs, **recomendamos utilizarla como versión de evaluación de las funcionalidades provistas por el componente, hasta tanto salga la versión estable definitiva**.
3. Esta versión del componente funciona con **Libertya 15.03**.
4. Para instalar el componente:
	1. Detener el servidor de aplicaciones.
	2. Copiar el jar dentro del directorio ServidorOXP/lib/plugins.
	3. Ejecutar Configurar.sh y guardar los cambios sin realizar modificaciones (este simplemente se realiza a fin de propagar el nuevo jar).
	4. Acceder al sistema mediante System Administrator, y ejecutar Instalador de Componentes y Plugins.
	5. Seleccionar el jar copiado en el segundo paso y clickear en Instalar.
	6. Salir de la aplicacion e iniciar el servidor.
5. Una vez instalado el componente, se debe crear el rol **Manufactura** con acceso al Árbol Primario de Menú **Manufactura** y asignar este rol a los usuarios que se considere pertinente.
6. Lea atentamente la documentación del componente.
7. Por cualquier consulta escriba a libertya@geneos.com.ar.
